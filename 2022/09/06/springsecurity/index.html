# 2-1 JWT 概述

  • 有状态与无状态比较
  • 无状态登录流程
  • JWT 的 Token 串生成

JWT 全称是 Json Web Token 是 JSON 风格轻量级的授权和身份认证规范

JWT 的 Token 串由三部分组成

  • header 头信息 - 采用 base64 编码生成 -- 类型与生成时间
  • Payload 载荷 - 用户身份信息，过期时间，签发人 - 采用 base64 编码
  • Signature 签名 是整个数据的认证信息 - header+Payload + 密钥 secret-RSA 非对称加密技术生成
